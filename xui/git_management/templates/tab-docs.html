Git Management

The Git Management extension allows you to export content from CloudBolt to a Git Repo. This extension is designed to work with GitHub and GitLab.

Initial Setup

1. Gather a Personal Access Token for your user account from GitHub or GitLab. The token must have the appropriate permissions to read the Git repo and create commits in the Git Repo.
     GitHub required permissions: read, write on the repo in question
     GitLab required permissions: api, read_api

2. Create a Git Management XUI configuration for each Git Repo you want to export content to. The name of the configuration will be used to identify the configuration in the Git Management XUI.
